Property detail
isReadyproperty
isReady:Boolean  [read-only]

Returns true when the TickedMessageManager is ready to begin sending and receiving messages. This will happen after run() is called, but not necessarily immediately, depending on any network handshaking that needs to happen.

Method detail
canSendMessage()method
public function canSendMessage():Boolean

Returns
Boolean — true if a call to sendMessage() will succeed. Games generally don't need to be concerned with this function; it should only return false if the TickedMessageManager is overloaded with too many pending messages.
getNextTick()method 
public function getNextTick():Array

Returns
Array — an Array containing the messages that were received during the oldest unprocessed tick, or null if there are no unprocessed ticks remaining.
run()method 
public function run():void

Should be called when the the TickedMessageManager should start listening for and and processing game ticks. In a multiplayer game, this is usually immediately after the GAME_STARTED event is received.

public function stop():void
