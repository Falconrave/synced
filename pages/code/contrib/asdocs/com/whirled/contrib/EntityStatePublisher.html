Classpublic class EntityStatePublisher
InheritanceEntityStatePublisher EntityStateListener flash.events.EventDispatcher

It is common for an item in a room to persist a memory that it wants to always be shared in its environment. While this is not a difficult operation, it can clutter up an otherwise tiny script. This class tries to offload the cryptic incantations as much as possible. Typical use is this: _publisher = new EntityStatePublisher(_control, KEY, false, stateUpdated); where _control is any EntityControl, KEY is the unique identifier used for storing the memory persistently and sharing in the room properties, and stateUpdated is an optional method that's called when the room state changes: this method should take a String argument (the key) and an Object (the value). To modify the stored state, use _publisher.setState(newState); and the current state is always available as _published.state
