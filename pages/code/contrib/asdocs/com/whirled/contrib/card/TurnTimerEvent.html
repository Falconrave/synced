Constant detail
EXPIREDconstant
public static const EXPIRED:String = "turntimer.expired"

Event type for when a turn has expired. For this event type, the time property is not used. The player property refers to the player whose turn has expired.

STARTEDconstant 
public static const STARTED:String = "turntimer.started"

Event type for when the timer has just kicked off. For this event type the time property is the amount of time expected to pass before the expiry event. The player property refers to the player being timed.
