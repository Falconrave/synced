Constructor detail
DataPackLoader()constructor
public function DataPackLoader(definitions:Array, loaded:Function = null, done:Function = null)

This constructor takes a list of pack definitions, and two optional callback functions. Pack definitions should be a list of objects, in the format produced by WhirledGameControl.getItemPacks() or WhirledGameControl.getLevelPacks().

The loader will extract the URLs of each content pack, and start loading them as data packs. Every time a data pack is finished processing, if the loaded callback is specified, it will be called, passing it the newly processed pack. Finally, after all packs have been processed, if the done callback is specified, it will be called with an array of all DataPack instances.

Parameters
definitions:Array — Array of content pack definitions.
 
loaded:Function (default = null) — Optional callback that will be called once for each loaded data pack. It should have the following signature: function (pack :DataPack) :void {}
 
done:Function (default = null) — Optional callback that will be called after all data packs were processed. It should have the following signature: function (packs :Array) :void {}
