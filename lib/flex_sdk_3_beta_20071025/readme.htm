Samples

The SDK samples directory contains source code for explorer sample application.

To compile the explorer application:

Windows: Run the build.bat file in the samples/explorer directory.
Macintosh or Unix: Run the build.sh file in the samples/explorer directory.

After you compile the explorer sample, open the explorer.html file in a browser to view it.

You can download additional Flex samples from the Adobe Flex Developer Center.
